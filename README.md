# Real-Time Collaborative Document Editor

A full-stack web application that allows multiple users to edit a document simultaneously with real-time synchronization and a live chat sidebar.

## Features
- **Real-Time Sync**: Live editing across multiple sessions using Socket.io.
- **Rich-Text Toolbar**: Support for Typography (Font, Size, Color), Styling, and Alignment.
- **Room Logic**: Unique URL generation for different document sessions.
- **Chat Sidebar**: Collaborative chat for users within the same document room.
